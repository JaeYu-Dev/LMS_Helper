# Project Portfolio

This directory stores planned and active engineering projects with their technical scope, implementation roadmap and measurable success criteria.

## Projects

### [HardDraft / Mini-Taalas LLM Accelerator](./harddraft-mini-taalas/README.md)

A staged FPGA-to-ASIC project for accelerating quantized language-model inference. The project focuses on speculative decoding, accepted tokens per second, fixed-backbone experiments, programmable adapters and eventual integration of a custom accelerator ASIC on a dedicated PCB.

**Status:** Planned

**Focus areas:** LLM inference, quantization, FPGA, RTL, ASIC, speculative decoding, embedded systems and PCB design.
